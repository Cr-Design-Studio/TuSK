# Welcome to TuSK Docs!

## Installation
Prerequisets:
- Python 3.9 or 3.9+
- Python set to path

1. Download the ZIP off the [Github](https://github.com/TutlaMC/tusk)
2. Unzip it to a folder
3. Run `python -m pip install -r requirements.txt`
